A development release.

A release candidate.

projector.VERSION = Version(major=0, minor=1, patch=0, release=Release(kind=<ReleaseKind.DEV: 'dev'>, n=0))¶

class projector.Version¶

Version information.

major¶

The major version.

This is X in X.Y.Z.

minor¶

The minor version.

This is Y in X.Y.Z.

patch¶

The patch numner.

This is Z in X.Y.Z.

release¶

The release information.

This will be None if and only if it is a released version.

to_short() → str¶

Return a short (X.Y) version string.
