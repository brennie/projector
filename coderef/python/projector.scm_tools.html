projector.scm_tools¶

SCM tool utilities.

projector.scm_tools.get_scm_tools¶

Return registered SCM Tools.

Supported SCM tools are registered with the projector.scm_tools entry point. The results will be cached for future use.

Returns:The registered SCM tools.
